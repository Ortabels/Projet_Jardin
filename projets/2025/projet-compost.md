## 2. Contexte
- Le terrain produit déjà des déchets végétaux (tailles, mauvaises herbes, foin…).
- Les habitants génèrent du compostage domestique utilisable pour la micro-ferme.
- Le compost nourrit les arbres, potager, buttes, et réduit les déchets municipaux.
- Activité cohérente avec l’objet de l’association : pédagogie, agroécologie.

---

## 3. Objectifs
- Créer une zone propre, structurée et durable de compostage.
- Réduire les déchets verts, recycler les ressources naturelles.
- Former les familles / enfants aux cycles naturels.
- Produire du compost pour les arbres, buttes et maraîchage.

---

## 4. Description du projet
- **Localisation** : zone nord/sud de la parcelle.
- **Type de compostage** :
  - 3 bacs bois (déchets verts / structurant / maturation)  
  - Aire de stockage du broyat
- **Matériaux** :
  - bois non traité, visserie galvanisée, charnières
- **Étapes** :
  1. Délimitation zone
  2. Construction 3 bacs  
  3. Mise en place signalétique  
  4. Mise en route + ateliers
- **Contraintes** :
  - stabilité du sol  
  - proximité d’une source de broyat (commune, particulier…)
